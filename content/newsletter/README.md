# 킵라인 레터 운영 가이드

## 파일 구성

- `welcome.md` — 구독 직후 자동 발송하는 환영 메일
- `001.md`, `002.md`, … — 주간 레터 원고 (발행 순서대로 번호)

## 스티비(Stibee) 첫 세팅 체크리스트

1. [stibee.com](https://stibee.com) 가입 → 무료 플랜으로 시작 (구독자 500명·월 4회 발송까지 무료)
2. **주소록 만들기** — 이름: `킵라인 레터`
   - 커스텀 필드 추가: `source`(텍스트), `subscribed_at`(텍스트) — 내보내기 CSV의 컬럼과 매칭
3. **발신자 정보** — 발신자 이름 `킵라인`, 발신 이메일 keepline1717@gmail.com 인증
4. **자동 이메일** — "구독 즉시 발송" 트리거로 `welcome.md` 내용 등록
5. **수신동의 설정** — 광고성 정보가 아니어도 해지 링크는 자동 삽입됨. 나중에 유료 상품 홍보를 넣게 되면 제목에 `(광고)` 표기가 법적 의무이니 그 전까지는 정보성 콘텐츠만 유지
6. 첫 발송 전 **본인 메일로 테스트 발송** 필수

## 주간 발행 루틴 (추천)

1. `npm run export:leads` → `exports/` 폴더의 CSV를 스티비 주소록에 업로드 (중복은 스티비가 자동 처리)
2. 이번 화 원고를 스티비 에디터에 붙여넣기 → 테스트 발송 → 확인 후 발송
3. 받은 답장 확인 → 좋은 질문은 다음 화 소재 메모

## 매호 형식 (랜딩 페이지에 약속한 그대로)

1. **몸의 신호** — 식욕·컨디션에서 요요가 시작되는 신호 한 가지
2. **지키는 루틴** — 단백질·근력·수면에서 이번 주 실천할 것 한 가지
3. **진료실 질문** — 다음 진료 때 그대로 쓸 수 있는 질문 한 개

규칙: 읽는 데 3분 이내 · 빼는 법 얘기 금지 · 매호 끝에 답장 유도 한 줄 · 의료 면책 문구 필수.
